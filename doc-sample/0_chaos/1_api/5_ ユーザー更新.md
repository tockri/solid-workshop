# （管理者用）ユーザー追加API

ユーザーを追加または更新する

## エントリポイント

PATCH /users/{id}

## リクエストボディ

(multipart-form-data)

| キー      | 型     | 必須 | 説明               |
| --------- | ------ | ---- | ------------------ |
| accountId | string | no   | 他と被らない文字列 |
| password  | string | no   | 任意の文字列       |

## レスポンス

| ステータス      | 説明                            |
| --------------- | ------------------------------- |
| 200 OK          | ログイン成功                    |
| 400 Bad Request | accountIdまたはpasswordが空     |
| 404 Not Found   | idが存在しない                  |
| 409 Conflict    | accountIdが他のレコードと被った |
