# （プレイヤー用）ログインAPI

accountIdとpasswordが一致していたら200 OKを返す。
その際、userテーブルのlastLoginを更新する。

## エントリポイント

POST /users/login

※ これはサンプルのため、ログイン機能やセキュリティは省略している。パスワードはDBに平文で登録されていて、単にリクエストと一致したら成功する。

## リクエストボディ

(multipart-form-data)

| キー      | 型     |
| --------- | ------ |
| accountId | string |
| password  | string |

## レスポンス

| ステータス    | 説明         |
| ------------- | ------------ |
| 200 OK        | ログイン成功 |
| 403 Forbidden | ログイン失敗 |

## レスポンス例

```
{
    "id": 1,
    "accountId": "alice",
    "lang": "jp",
    "score": 1200
}
```
