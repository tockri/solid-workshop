# （管理者用）ユーザーID一覧を取得するAPI

## エントリポイント

GET /users

※ これはサンプルのため、ログイン機能やセキュリティは省略している。管理者以外が呼べないようにする等の考慮はしない。

## レスポンス例

```
[
    {
        "id": 1,
        "accountId": "alice",
        "lastLogin": "2025-10-10T10:00:00"
    },
    {
        "id": 2,
        "accountId": "bob",
        "lastLogin": "2025-10-10T10:00:00"
    }
]
```
