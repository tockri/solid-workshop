# （管理者用）ユーザー追加API

ユーザーを追加または更新する

## エントリポイント

POST /users

## リクエストボディ

(multipart-form-data)

| キー      | 型     | 必須 | 説明               |
| --------- | ------ | ---- | ------------------ |
| accountId | string | yes  | 他と被らない文字列 |
| password  | string | yes  | 任意の文字列       |

## レスポンス

| ステータス      | 説明                             |
| --------------- | -------------------------------- |
| 200 OK          | ログイン成功                     |
| 400 Bad Request | 必須のパラメータがない、または空 |
| 409 Conflict    | accountIdが他のレコードと被った  |
